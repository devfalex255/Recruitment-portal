# Warehouse Backend

## Project Structure
1. `wemis_accounts`:  
    User managements, Separation and profile managements. Hapa utaasign userkulingana na nafasi zao. Mfano: DG, warehouse managers, Staff na users wengine wenye role ya kufanya monitoring ya Mfumo na reports.

1. `wemis_settings`:  
    Settings zote za mfumo zinafanyika hapa. kitu chochote ambacho ni confugurable kinafanika hapa. Mfano: Locations za warehouse na Settings nyingine.

1. `wemis_auth`:  
    Roles na permission zinakuwa managed huku. Kuanzi kutengeneza roles, ku-assing role kwa users na kuzifuta.

1. `wemis_warehouse`:  
    Management ya warehouse yote inafanyika huku. Kutenegeneza warehouse, building, zones na sensor assignments na registations inafanyika huku.

1. `wemis_reports`:  
    Report generations zote zinafanyika huku. report za Warehouse, Building, zones na sensor zote zinatokea huku. Pamoja na report nginenezo. Hii module pia inatoa dashboards and Analysis
